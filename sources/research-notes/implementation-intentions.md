# Implementation intentions

## Scope

This source note supports cautious reader-facing content about implementation intentions as if-then plans for action initiation, self-control, habit support, and decision-process safeguards.

## Executive summary

- Implementation intentions are commonly described as plans that link a specified future situation to a specified response: if situation X occurs, then I will do Y.
- They differ from goal intentions: goal intentions state desired outcomes; implementation intentions specify when, where, or how to act.
- Research by Gollwitzer and colleagues supports implementation intentions as a goal-striving strategy in many studied contexts.
- Practical use requires concrete cues and concrete feasible actions.
- Implementation intentions are not万能: vague cues, too many rules, large actions, changing contexts, or safety-critical situations can limit usefulness.

## Key claims

### Claim 1: Implementation intentions specify a situation-response link.

Evidence:

- Gollwitzer's 1999 American Psychologist article describes implementation intentions as plans that link anticipated critical situations to goal-directed responses.
- Gollwitzer and Brandstätter's 1997 work describes the format of intending to do a specified action when a specified situation is encountered.

Uncertainty / limits:

- This note does not extract effect sizes.
- Reader-facing content should not imply that any if-then sentence will work equally well.

### Claim 2: Implementation intentions are distinct from goal intentions.

Evidence:

- Gollwitzer's work distinguishes deciding on a desired end state from specifying situational cues and responses that support goal pursuit.
- The practical distinction maps well to Commonplace: “判断を改善したい” is a goal intention; “もし反対根拠を一つも言えないなら、判断を記録してから実行する” is an implementation intention.

Uncertainty / limits:

- Some goals require skills, resources, or external support beyond planning.
- If-then plans do not replace domain knowledge.

### Claim 3: Implementation intentions can support action initiation and self-regulation, but context matters.

Evidence:

- Reviews and encyclopedia entries consistently describe implementation intentions as goal-striving/self-regulation tools.
- The literature includes applications to action initiation, habit-like responding, and health/self-control behavior, but transfer to each Commonplace domain must remain cautious.

Uncertainty / limits:

- Effects vary by goal commitment, cue detectability, action feasibility, and context stability.
- Plans can become brittle when the environment changes.
- In high-risk medical, legal, financial, or interpersonal safety situations, self-made rules are not enough.

## Practical implications

- For judgment pages, if-then plans should be short, concrete, and observable.
- Useful patterns include: if strong emotion appears, delay; if no opposing evidence is written, pause; if invalidation condition is hit, review rather than improvise.
- Use implementation intentions as a bridge between pre-decision checks and judgment rules.

## Safe wording

- 「実行意図は、『もしXが起きたらYをする』という形で、行動のきっかけと反応を事前に結びつける方法です。」
- 「目標を決めるだけでなく、いつ・どこで・何をするかを具体化します。」
- 「実行意図は行動を始めやすくする可能性がありますが、万能ではありません。」
- 「条件と行動が曖昧な場合、または安全リスクがある場合は、短い自己ルールだけで処理しません。」

## Cautions

- Do not promise habit formation or outcome improvement.
- Do not use if-then rules to bypass professional advice or risk controls.
- Do not create too many conditions.
- Do not define actions that are too large, vague, or unsafe.

## Needs verification

- Direct extraction of Gollwitzer 1999 full text.
- Direct extraction of Gollwitzer and Brandstätter 1997 full text.
- Meta-analyses and boundary-condition reviews before citing quantitative effects.
- Domain-specific evidence before claiming effects for trading, writing, emotion regulation, or career decisions.
