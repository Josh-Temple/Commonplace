## Public-safe template

```text
Date/session label:
Setup type:
Context summary:
Structure summary:
Level used:
Volume Profile observation:
Scenario before action:
Trigger observed:
Invalidation condition:
Risk/reward quality:
Emotion before action:
Decision: Entry / Wait / No trade
Process grade:
Rule adherence:
Exit quality:
One lesson:
One adjustment:
```

## Do not include

- Account size
- Position size
- Exact P/L
- Broker screenshots
- Private diary entries
- Personally identifying information

このtemplateはprocess review用であり、個別の成績公開や金融助言ではありません。
