## 使い方

このページは、長く読むためではなく、必要な場面で1枚だけ読むための実践カード集です。

強い症状、安全リスク、現実感の低下、自傷・他害リスク、躁的高揚、強い解離がある場合は、このページではなく安全確保と専門的支援を優先します。

医療、心理療法、投資助言の代替ではありません。個人的な取引記録、日記、口座情報、損益、個人情報は公開リポジトリに入れません。

## カード1：5分リセット

1. 足裏を感じる。
2. 呼吸を変えずに、今の呼吸に気づく。
3. 身体の緊張を一か所だけ見る。
4. 感情に短い名前をつける。
5. 次の小さい行動を一つ選ぶ。

## カード2：怒りが強いとき

1. 送信しない。言い返さない。決めない。
2. 手、胸、顎、肩の力みを見る。
3. 「怒り」「正当化」「論破したい思考」と名づける。
4. 守りたい価値を一つ選ぶ。
5. 10分後に読み直す前提で、一文だけ下書きする。

## カード3：不安が強いとき

1. 未来予測が始まっていると気づく。
2. 足裏か手の接触へ注意を置く。
3. 「不安」「予測」「準備したい衝動」と名づける。
4. 今できる最小行動を一つ選ぶ。
5. 1分だけ実行する。

## カード4：自己批判が強いとき

1. 「自分はだめだ」ではなく、「自己批判の思考が出ている」と言う。
2. 肩、胸、腹、顔の感覚を見る。
3. 反論しようとしすぎない。
4. 回復、健康、学習、誠実さのどれかを選ぶ。
5. 水を飲む、5分休む、1行書く、相談文を下書きする。

## カード5：呼吸が苦しいとき

1. 呼吸を対象にしない。
2. 目を開ける。
3. 足裏、手の接触、部屋の音、見える物を対象にする。
4. 30秒だけ外部対象を確認する。
5. 悪化するなら中止する。

## カード6：寝る前

1. 今日を評価しすぎない。
2. 布団の接触、重さ、温度を見る。
3. 思考が出たら「計画」「反省」「心配」と名づける。
4. 呼吸を深くしようとしない。
5. 眠れなければ、練習を成功させようとせず休む。

## カード7：判断前の一呼吸

1. 今すぐ決める必要があるか確認する。
2. 身体の緊張を見る。
3. 感情に名前をつける。
4. 判断基準を一つだけ書く。
5. その基準に照らして、行動する・保留する・相談する、を選ぶ。

## カード8：トレード前・重要判断前

1. いま焦り、取り返したさ、退屈、恐怖がないか見る。
2. 身体反応を見る。
3. 「入りたい理由」と「入らない理由」を一つずつ言う。
4. ルールに合わなければ、見送る。
5. 見送ることも行動として扱う。

これは投資助言ではありません。重要判断の前に、感情と行動を分けるための一般的なカードです。

## カード9：短いメッタ

1. 無理に温かい気持ちを作らない。
2. まず安全な対象を選ぶ。
3. 「安全でありますように」「少し楽でありますように」と短く言う。
4. 難しい人へ急いで広げない。
5. 苦しくなったら足裏や部屋の音へ戻る。

## カード10：先延ばしから戻る

1. 「やりたくない」ではなく、「回避したい衝動がある」と言う。
2. 身体の重さ、胸、肩、目の疲れを見る。
3. 価値を一つ選ぶ。学習、健康、誠実さ、関係性など。
4. 2分だけできる行動に小さくする。
5. 終えたら成否ではなく「戻れた」ことを記録する。

## 注意点

- 呼吸、身体、感情に注意を向けて悪化する場合は、外部対象、移動、会話、休息へ切り替えます。
- 感情を消してから行動するのではなく、感情があっても安全な小さい行動を選びます。
- 強い症状や危険サインがある時は、カードではなく[[mindfulness-safety]]と専門的支援を優先します。
